# homebrew-neovim
### Wraps `neovim/neovim/neovim`` cask
Neovim, for me!

### Features
* Use XDG paths
* True color
* Use vim plug
* Fancy statusline
* gruvbox - great true color support
* Highlight EOL whitespace
* Autocomplete
* Commands
 * UpdatePlugins
 * SaveAndRun
 * CopyToClipboard (copy file to clipboard)
